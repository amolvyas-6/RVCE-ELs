# RVCE Engineering Projects Collection

A comprehensive collection of software engineering projects, laboratory exercises, and AI research developed as part of the RVCE curriculum.

## 🚀 Projects Overview

This repository is a central hub for various projects spanning AI/ML, Cloud Computing, DBMS, DAA, Operating Systems, and more.

| Project Name | Domain | Key Technologies | Description |
| :--- | :--- | :--- | :--- |
| **[UDAAN - AI Lawyer](./UDAAN%20-%20AI%20Lawyer)** | Legal Tech / AI | React, Express, FastAPI, LangChain, RAG | A comprehensive legal case management system with AI document analysis. |
| **[Cloud Computing](./Cloud%20Computing)** | Cloud / Microservices | FastAPI, React, RabbitMQ, MinIO, Docker | Wildlife detection AI powered by a microservices architecture. |
| **[AIML Lab](./AIML%20Lab)** | AI / Audio | PyTorch, Torchaudio, Librosa, uv | Audio deepfake detection using LFCC-LCNN and Transformers. |
| **[DBMS Lab](./DBMS%20Lab)** | Full-Stack / DBMS | React, Express, Prisma, FastAPI, Nixtla | Multi-tier data management app with integrated AI forecasting. |
| **[GestureX](./GestureX)** | CV / ML | OpenCV, Mediapipe, TensorFlow, Keras | AI-powered hand tracking and gesture-based computer control. |
| **[DAA Lab](./DAA%20Lab/SudokuSolver)** | Algorithms / GUI | C++, Qt, CMake, Backtracking | Sudoku solver and generator with step-by-step visualization. |
| **[OS Lab](./OS%20Lab)** | Systems | C, POSIX API, Linux Shell | Custom Unix shell implementation with piping and redirection. |
| **[Python Projects](./Python)** | Simulation / Data | Pandas, Numpy, Matplotlib | Orbital mechanics simulator and planet distance calculator. |

## 🛠️ General Tech Stack

- **Frontend**: React (18/19), Vite, Tailwind CSS (v3/v4), Shadcn/ui
- **Backend**: Node.js (Express), FastAPI (Python), C (Systems)
- **AI/ML**: PyTorch, TensorFlow, Keras, Mediapipe, LangChain, Nixtla
- **Database**: PostgreSQL (Prisma), MongoDB (Mongoose), MinIO (S3), Redis
- **DevOps**: Docker, Docker Compose, Caddy, uv, npm

Each project is self-contained with its own set of dependencies and execution instructions. Please navigate to the individual project directories to find their specific `README.md` files for setup and running instructions.
