# DAA Lab - Design and Analysis of Algorithms

This directory contains laboratory work and projects related to algorithmic design, performance analysis, and problem-solving techniques.

## 🚀 Projects Overview

- **[Sudoku Solver & Generator](./SudokuSolver)**: A C++/Qt application that uses a backtracking algorithm to generate and solve Sudoku puzzles with real-time visualization.

## 📂 Project Structure

- `SudokuSolver/`: Core project with CMake-based build system and Qt GUI.
- `DAA-Lab Part B-Report.pdf`: Detailed project report and analysis.
- `literature paper.pdf`: Research papers related to the Sudoku-solving algorithm.

## 🛠️ Tech Stack

- **Language**: C++ (C++17/20)
- **Framework**: Qt (GUI)
- **Build System**: CMake
- **Algorithm**: Backtracking
